# Customer-Segmentation-using-Clustering

Create a clustering system to:

Group similar customers based on purchasing behavior

Machine should automatically identify:

High-value customers
Medium customers
Low-value customers

without labels.

Problem Statement

A shopping mall wants to understand customer behavior.

They have customer information like:

Age
Annual Income
Spending Score

But they don’t know:

“Which customers belong to which category?”

Your task:

Use clustering techniques to group customers automatically.

Dataset Features

Student should create or use dataset containing:

Feature	Description
Customer ID	Unique ID
Age	Customer age
Gender	Male/Female
Annual Income	Income per year
Spending Score	Shopping behavior
Tasks to Perform
Task 1: Data Understanding

Analyze:

Number of rows
Number of columns
Missing values
Data types

Questions:

Which feature is numerical?
Which feature is categorical?
Task 2: Data Visualization

Create visual analysis.

Suggested graphs:

Scatter Plot

Income vs Spending

Goal:

See customer distribution.

Histogram

Income distribution.

Pairplot

Relationship between features.

Task 3: Apply K-Means Clustering ⭐

Perform clustering using:

Features:
Income
Spending Score
